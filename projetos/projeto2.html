API de Previsão do Tempo

Airflow + pandas + PostgreSQL

⬅ Voltar

Descrição do Projeto

Este projeto realiza a extração semanal de previsões do tempo dos próximos 15 dias utilizando uma API pública. As previsões são armazenadas em um banco PostgreSQL para análise e visualização futura.

Stack Utilizada

  • Apache Airflow para agendamento semanal
  • API pública de clima (Ex: OpenWeatherMap)
  • pandas para manipulação dos dados
  • PostgreSQL para armazenamento
