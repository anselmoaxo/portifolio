Descrição do Projeto

Projeto de pipeline completo para ETL de dados de uma concessionária de veículos. Ele envolve a extração de dados operacionais de vendas, clientes, produtos e categorias de um banco de produção, o carregamento para um Data Warehouse e a transformação dos dados com dbt.

Stack Utilizada

  • Apache Airflow para orquestração
  • PostgreSQL como banco de origem e destino
  • dbt para transformação (camadas: staging, dimensions, analytics)
  • Power BI para visualização dos dados
